개인정보 처리방침

시행일: 2026년 7월 18일

로아담(이하 "앱")은 이용자의 개인정보를 소중히 여기며, 관련 법령을 준수합니다. 본 방침은 앱이 어떤 정보를 어떻게 처리하는지 설명합니다.

1. 수집하는 정보

앱은 회원가입·로그인 없이 사용하며, 이름·이메일·전화번호 등 개인 식별 정보를 수집하지 않습니다. 서비스 제공을 위해 처리되는 정보는 다음과 같습니다.

항목용도보관
검색한 게임 캐릭터명 로스트아크 공식 API 를 통한 게임 정보 조회 서비스 서버에 조회 결과를 일시 캐시 (최대 1시간) 후 자동 갱신·삭제
등록한 원정대/캐릭터 목록, 앱 설정 내 캐릭터 관리, 화면 표시 이용자 기기에만 저장 (앱 삭제 시 함께 삭제)
푸시 토큰 (알림 설정 시) 모험섬 일정·보석 가격 알림 발송 Firebase Cloud Messaging 에서 관리, 알림 해제/앱 삭제 시 무효화
앱 사용 통계 (기기 모델, OS 버전, 화면 이용 기록 등 비식별 정보) 서비스 품질 개선 Google Firebase Analytics 정책에 따름

2. 제3자 서비스

앱은 서비스 제공을 위해 아래 외부 서비스를 이용하며, 각 서비스의 개인정보 정책이 적용됩니다.

  • Google Firebase — 사용 통계 분석, 푸시 알림
  • Cloudflare — API 서버 인프라
  • 로스트아크 오픈 API (Smilegate RPG) — 게임 데이터 조회

3. 개인정보의 제공 및 판매

앱은 이용자의 정보를 제3자에게 판매하거나 마케팅 목적으로 제공하지 않습니다.

4. 이용자의 권리

  • 기기 내 저장 데이터는 앱 삭제로 즉시 완전히 삭제됩니다.
  • 푸시 알림은 앱 또는 시스템 설정에서 언제든지 해제할 수 있습니다.
  • 기타 개인정보 관련 요청은 아래 연락처로 문의할 수 있습니다.

5. 아동의 개인정보

앱은 아동을 대상으로 하지 않으며, 아동의 개인정보를 고의로 수집하지 않습니다.

6. 방침의 변경

본 방침이 변경되는 경우 이 페이지를 통해 공지하며, 중요한 변경은 앱 내에서 안내합니다.
